**Status:** Approved (UD-008)  
**Date:** 2026-07-19

## Decisions

1. **Files (`apps.files`):** `FileObject` metadata + local filesystem storage under `MEDIA_ROOT/org/{org_id}/project/{project_id}/`. Project members download; project_admin/engineer upload. SHA-256 stored.
2. **Calculations (`apps.calculations`):** `CalculationMethod` registry + `CalculationRun` + `CalculationLog`. Stub sync executor for `platform_stub_v1` produces deterministic JSON results.
3. **Reporting (`apps.reporting`):** `ReportArtifact` linked to project/run; basic generator writes JSON summary file + FileObject.
4. **Ops:** Documented backup/monitor/rollback procedures; health remains liveness-only.
5. **R0 E2E:** Automated test covers auth → org → project → file → calc run → report.
